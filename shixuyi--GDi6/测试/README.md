# 测试 (Testing)

本项目中的文件归类及说明：

## benchmarks/ (基准测试)
项目路径: `data/benchmarks/`, `app/services/`
- xingtu_10_questions.json - 10问基准测试集
- benchmark_service.py - 基准测试服务

## seed_data/ (种子数据)
项目路径: `data/seed/`
- xingtu_manual_qa.json - 兴图新科人工标注QA

## test_scripts/ (测试脚本)
项目路径: `scripts/`
- run_benchmark.py - 运行基准测试

## comparison/ (对比分析)
项目路径: `scripts/`
- compare_answers.py - 答案对比分析脚本
