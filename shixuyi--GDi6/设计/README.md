# 设计 (Design)

本项目中的文件归类及说明：

## frontend/ (前端静态)
项目路径: `static/`
- index.html - 前端入口页面
- css/style.css - 样式表
- js/app.js - 前端交互逻辑

## data_design/ (数据结构)
项目路径: `data/processed/`
- cleaned_upload.pdf - 清洗后PDF
- cleaned_upload.txt - 清洗后文本
- document_registry.json - 文档注册表
- xingtu_chunks.json - 兴图新科chunk数据

## raw_data/ (原始数据)
项目路径: `data/raw/`, `data/uploads/`
- 招股说明书PDF - 原始招股书
- 上传的各种PDF文档
