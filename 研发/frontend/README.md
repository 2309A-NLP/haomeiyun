# 法律RAG系统前端

这是一个为法律智能问答系统设计的现代化、美观前端页面。

## 最新更新

- ✅ 移除了医生和AI陪伴者角色，专注于法律咨询
- 🎨 全面美化界面，采用现代化设计风格
- ✨ 添加动画效果和响应式布局
- 📱 优化移动端体验

## 功能特性

- 🎨 现代化UI设计，采用渐变背景和毛玻璃效果
- 👥 支持专业法律角色选择（5个法律专业角色）
- 💬 实时对话界面，支持流式回答
- 📚 显示法律引用来源和推荐追问
- ⚠️ 风险等级提示
- 📱 完全响应式设计，支持移动端
- ✨ 流畅的动画效果和交互体验

## 可用角色

- 综合律师 - 全面法律咨询
- 刑事辩护律师 - 刑事案件专业咨询
- 劳动法律师 - 劳动纠纷专业咨询
- 婚姻家事律师 - 婚姻家庭问题咨询
- 合同律师 - 合同纠纷专业咨询

## 界面特色

- **渐变背景**：紫色渐变背景营造专业氛围
- **毛玻璃效果**：现代化的半透明对话框
- **消息气泡**：带箭头的气泡样式，区分用户和AI消息
- **动画效果**：消息出现动画和按钮悬停效果
- **自定义滚动条**：美化的滚动条样式
- **响应式布局**：完美适配各种屏幕尺寸

## 使用方法

1. 启动后端服务：
   ```bash
   python run.py
   ```

2. 打开浏览器访问：http://localhost:8001

3. 开始使用：
   - 选择合适的法律专业角色
   - 输入用户ID
   - 提出法律问题
   - 查看AI回答、引用来源和推荐追问

## 技术栈

- HTML5 + CSS3 (现代化样式和动画)
- JavaScript (ES6+)
- Bootstrap 5.3 (响应式框架)
- Font Awesome 6.0 (图标库)
